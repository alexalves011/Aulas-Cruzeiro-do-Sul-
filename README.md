# Aulas-Cruzeiro-do-Sul

![Status](https://img.shields.io/badge/status-em_andamento-blue)

Repositório pessoal para centralizar materiais de estudo, exercícios e projetos da minha graduação na **Universidade Cruzeiro do Sul**. Este espaço documenta minha jornada acadêmica, abrangendo conteúdos de diversos semestres e disciplinas.

## 🎯 Objetivos

- Organizar e preservar materiais didáticos e projetos desenvolvidos durante o curso.
- Demonstrar evolução nas habilidades técnicas e conceituais ao longo dos semestres.
- Servir como portfólio acadêmico para referência futura e compartilhamento de conhecimento.

## 📚 Estrutura Acadêmica

### 1º Semestre

- **Engenharia de Software**: Materiais e atividades relacionadas à engenharia de software.
- **Programação Web**: Projetos e exemplos de desenvolvimento web, incluindo HTML, JavaScript e CSS.
- **Sistemas Operacionais**: Anotações e trabalhos sobre sistemas operacionais.
- **Técnicas de Desenvolvimento de Algoritmos**: Exercícios de lógica e algoritmos em Java.

### 2º Semestre

- **Análise e Projeto de Sistemas**: Estudos sobre análise e projeto de sistemas.
- **Banco de Dados**: Conceitos e práticas de bancos de dados.
- **Estrutura de Dados**: Fundamentos de estruturas de dados.
- **Técnicas de Programação**: Técnicas avançadas de programação.
- **Itinerário Extensionista 1**: Projetos extensionistas e atividades práticas.
- **Programação Orientada a Objetos**: Conceitos de POO e aplicações práticas.

## 💻 Tecnologias Utilizadas

As principais linguagens e tecnologias empregadas nos projetos incluem:

- **Java**: Para desenvolvimento de algoritmos e aplicações.
- **HTML**: Estruturação de páginas web.
- **JavaScript**: Interatividade e lógica no front-end.
- **CSS**: Estilização de interfaces web.

## 📝 Controle de Atividades

O arquivo `Tarefas Cruzeiro do Sul.xlsx` é utilizado para o controle geral de entregas, prazos e atividades acadêmicas.

## 📌 Considerações Finais

Este repositório reflete meu compromisso com o aprendizado contínuo e o desenvolvimento profissional. À medida que avanço nos semestres, novos conteúdos e projetos serão adicionados para manter uma visão abrangente da minha trajetória acadêmica.
